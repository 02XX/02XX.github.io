反正我使用官方提供的方法不能显示图片

我在网上找到如下方法解决：

  1. 下载使用hexo-assert-image

npm install hexo-assert-image

  1. 更改_config.yml

将post_asset_folder:false改为
post_asset_folder:true

当新建文档时会在_post文件夹中自动生成一个同名的资源文件夹，这个文件夹可以放入图片，方便以后引用。
